Что такое CMS

CMS - "система управления контентом (двидок) - написанная РНР-программистами основа для сайта, с помощью которой вы сожете управлять сайтом (добавлять контент, менять пункты меню и т.п.) не зная HTML и CSS"

Однако, для того чтобы сделать всесь сайт с помощью CMS потребуются услуги и программиста, и дизайнера, и верстальщика. И капиталовложения.

Какие бывают cms

Бывают различные системы управления контентом: для инетнет-магазинов, для блогов, для форумов и т.д.

Примеры csm

Примеры популярных CMS: Joomla, WordPress (для блогов), PhpBB (для форумов)
